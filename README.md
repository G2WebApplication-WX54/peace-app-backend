# Peace App

## Summary
Peace App API Application, made with Microsoft C#, ASP.NET Core, Entity Framework Core and MySQL persistence.

## Features
- RESTful API
- OpenAPI Documentation
- Swagger UI
- ASP.NET Framework
- Entity Framework Core
- Audit Creation and Update Date
- Custom Route Naming Conventions
- Custom Object-Relational Mapping Naming Conventions.
- MySQL Database
- Domain-Driven Design
